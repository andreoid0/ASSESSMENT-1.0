HTML & CSS Quick-Reference Guide

HTML Tags

Below are some commonly used HTML tags and their descriptions:

  • <h1> - <h6>: Defines headings from largest to smallest.
  • <p>: Defines a paragraph.
  • <a href="">: Creates a hyperlink.
  • <img src="">: Embeds an image.
  • <ul> & <ol>: Creates unordered and ordered lists.
  • <table>: Defines a table.
  • <div>: Defines a container for styling.
  • <span>: Defines inline elements.
  • <form>: Defines a form for user input.
  • <input>: Creates input fields.

CSS Properties

Here are some common CSS properties used for styling:

  • color: Sets the text color.
  • background-color: Sets the background color.
  • font-size: Defines the size of the text.
  • margin: Sets the space outside an element.
  • padding: Sets the space inside an element.
  • border: Defines the border of an element.
  • display: Specifies the display behavior of an element.
  • position: Specifies the positioning method of an element.
